Windows:
install MSYS2:
winget install -e --id MSYS2.MSYS2

Open “MSYS2 MINGW64” from Start Menu.
Update + install tools/libraries:
pacman -Syu
pacman -S --needed base-devel mingw-w64-x86_64-toolchain mingw-w64-x86_64-raylib


because the MSYS2 console sucks, you can still use powershell like that (back in powershell):
$env:PATH="C:\msys64\mingw64\bin;C:\msys64\usr\bin;" + $env:PATH
 
 test if it worked:
 gcc --version

Compile your program:
gcc radar.c -o radar.exe -lraylib -lopengl32 -lgdi32 -lwinmm
./radar.exe


To include raylib.h in vsc:

in vsc press ctrl + shift + p. 
Type C/C++: Edit Configurations (UI)
set settings like this:
Compiler path → point to the compiler you actually use (example for MSYS2):
C:\msys64\mingw64\bin\gcc.exe
IntelliSense mode → windows-gcc-x64
Include path → add your include folders, commonly:
${workspaceFolder}/**
C:\msys64\mingw64\include
C:\msys64\mingw64\x86_64-w64-mingw32\include

(and any library-specific include folder, e.g. raylib if it’s somewhere else)

Save (it writes .vscode/c_cpp_properties.json)
Press Ctrl+Shift+P → C/C++: Reset IntelliSense Database (or just reload window)
